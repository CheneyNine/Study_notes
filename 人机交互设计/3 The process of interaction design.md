- 让用户参与的重要性
- 用户参与度的度量
- 以用户为中心的方法是什么
- 四个交互设计的基本活动
- 交互设计的生命周期模型
一些实际的问题：
- 用户是什么
- 用户的需求是什么
- 如何生成可替代的设计
- 如何从多个设计中选择
- 如何使用生命周期模型生成交互设计


# 双菱形设计
1. 发现问题
2. 定义关注的领域
3. 潜在的解决方案
4. 交付可行的方案

# 以用户为中心的方法
1. 用户是谁？了解用户认知、人格、态度等，研究用户的使用环境
2. 以经验为基础的测量
3. 交互设计，出现问题后对其进行修复

# 交互设计的四个步骤
1. 发现需求
2. 设计多个选择、备选方案
3. 原型构建
4. 评估产品和用户体验
